# Documentation Index

This directory contains the stable process, report, and deliverable documents for the pilot. Routine run outputs belong under `outputs/runs/<run_id>/`, not in `docs/`.

## Final Deliverables

- `final_presentation_slide_content.md`: presentation-ready slide content for the 12-minute talk.
- `acl_report_draft.md`: first full ACL-style report draft.
- `final_deliverable_evidence_table.md`: source-of-truth evidence table for completed examples.
- `professor_qna_prep.md`: concise Q&A preparation.

## Pilot Findings

- `pilot_mechanistic_findings_369.md`: example 369 mechanistic findings.
- `pilot_mechanistic_findings_813.md`: example 813 mechanistic findings.
- `two_example_pilot_comparison.md`: conservative comparison of examples 369 and 813.
- `excluded_examples_summary.md`: failed, parked, and ambiguous examples.

## Workflow and Registry

- `example_ingestion_and_registry_sop.md`: future-student SOP for example admission and lifecycle.
- `examples_registry_schema.md`: registry schema and field meanings.
- `run_artifact_workflow.md`: run-folder architecture and reproducibility workflow.
- `ragtruth_leads_policy.md`: policy for RAGTruth-derived leads.
